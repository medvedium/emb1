Нитки для машинной вышивки

Вышивальные нитки бывают разными по составу: полиэстер, вискоза, шерсть, хлопок, металлические нитки. Для машинной вышивки используются нитки, как из натуральных, так и из химических волокон. В качестве сырья для производства ниток используются полиэфирные волокна, вискоза, хлопок, шерсть и др.

В настоящее время наиболее распространены вискозные (состав 100% вискоза) и полиэфирные (состав 100% полиэстер) нитки. ВИСКОЗА — они отличаются эффектным внешним видом за счёт ровного шелковистого блеска, имеют самую богатую цветовую палитру — до 500 оттенков, включая меланжевые и многоцветные оттенки. По структуре — в два сложения, достаточно эластичные, но имеют низкую устойчивость к истиранию! Важно помнить, что их нельзя отбеливать! Наиболее перспективными сейчас представляются полиэфирные нитки — ПОЛИЭСТЕР-100%.
