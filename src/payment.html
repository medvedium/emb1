Онлайн оплата банковской картой

Вы можете оплатить заказ, если являетесь владельцем пластиковых карт:

После оформления заказа на сайте, дождитесь подтверждения от оператора о наличии товара на складе, затем совершите оплату.

Все платежи через электронные платежные системы осуществляются только по безопасному защищенному протоколу при поддержке ОАО «Петербургский социальный коммерческий банк» ПСКБ.

После выписки счета вы будете перенаправлены на сервер ПСКБ (который имеет все необходимые лицензии и сертификаты на проведение платежей через Интернет) для введения данных кредитной карты. Передача данных происходит в защищенном режиме, при котором полностью исключена возможность перехвата информации о вашей карте. Для защиты информации от несанкционированного доступа на этапе передачи от клиента на сервер системы ПСКБ используется протокол SSL 3.0, сертификат сервера выдан компанией Deiteriy - признанным центром выдачи цифровых сертификатов. Вы можете проверить подлинность сертификата сервера.

Запрещены платежи по карте, не принадлежащей лицу, указанному в качестве плательщика при оформлении заказа! Все возвраты по платежам по кредитной карте могут быть осуществлены только безналичным путем зачисления на карточный счет.

Частичная оплата бонусными баллами

С каждого оплаченного заказа 5% начисляется в виде бонусных баллов. Один бал- это один рубль. Баллами можно оплатить до 20% от стоимости следующего заказа, для этого вам нужно будет в личном кабинете при оформлении заказа использовать кнопку "оплатить бонусами". Стоимость заказа автоматически уменьшится на сумму использованных бонусов. Бонусы можно использовать в течении 3х месяцев, по истечении этого срока они обнуляются. Срок в три месяца начинается каждый раз с последнего заказа.
